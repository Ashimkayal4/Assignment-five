Important FAQ

What is the dom object Model (DOM)?

The DOM is a tree structure representing HTML or XML documents.

How do you select an element from the DOM?

Use `document.querySelector()` or `document.getElementById()` to select elements.

what is the event delegation in the context of the DOM ,and why is it useful ?

Event delegation assigns events to parent elements, improving performance efficiency.

How do you manipulate an element's attributes and styles using DOM?

Use `setAttribute()`, `getAttribute()`, or directly modify `style` property for manipulation.
